Web Pemodelan Pasut

Pemodelan Harmonik Pasang Surut

Upload data CSV/XLSX atau ambil data dari IOC, pilih sensor, atur rentang waktu, pilih resolusi 1 menit / 15 menit / 30 menit / 1 jam, lalu hasil observasi, model, residual, dan report TXT akan dibuat langsung di browser.

1) Sumber data

Format yang didukung: baris 1 = nama stasiun, baris 2 = nama kolom, baris 3 dst = data.

Isi kode stasiun, periode, dan tanggal akhir untuk membentuk URL otomatis, atau paste langsung URL IOC. Bila request langsung dari browser diblokir server IOC, website akan menampilkan pesan error dan mode upload file tetap bisa dipakai.

Belum ada data yang dipilih.

2) Pengaturan analisis

Default mengikuti script MATLAB: M2, S2, K1, O1, N2, K2, P1, Q1.
